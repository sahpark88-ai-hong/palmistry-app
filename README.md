# Palmistry AI

양손 손금 이미지를 업로드하거나 촬영해 엔터테인먼트 관점의 손금 해석 리포트를 보여주는 초기 UI 프로토타입입니다.

## 실행 방법

`index.html` 파일을 브라우저로 열면 바로 실행됩니다.

## 현재 구현

- 모바일 폭에서는 촬영 가이드형 UI로 표시
- PC/태블릿 폭에서는 양손 비교 대시보드 UI로 표시
- 기본 시작 화면은 PC에서도 폰 보기 미리보기로 표시
- 상단에서 자동 / 폰 보기 / PC 보기 전환 가능
- 왼손/오른손 이미지 선택 및 미리보기
- 모바일 촬영 버튼 클릭 시 카메라 촬영 / 갤러리 선택 패널 표시
- 왼손/오른손 선택 완료 썸네일과 다시 선택 버튼 표시
- 모바일 흐름을 촬영 준비 / 스캔 진행 / 결과 화면으로 분리
- 손금 라인 오버레이, 분석 진행 상태, 결과 카드 표시
- 의료/예언 목적이 아닌 엔터테인먼트 안내 문구 포함
